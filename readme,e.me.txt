# Web Automation Script

This repository contains a Python script that automates interactions with a web page using **Selenium**. The script performs the following tasks:

- Logs into a website.
- Handles CAPTCHA (if present).
- Uploads documents and submits forms.
- Automatically likes a post on the website.
- Automates multiple verification requests with random delays to mimic human behavior.

## Prerequisites

Before running the script, make sure you have the following installed:

### 1. **Python**:
   - Download and install Python from [python.org](https://www.python.org/downloads/).

### 2. **Selenium Library**:
   - Install the Selenium package using pip:
   ```bash
   pip install selenium
